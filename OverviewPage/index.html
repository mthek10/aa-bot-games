Automation Anywhere Challenge - Embedded Automation!

Challenge Background

Certain financial transactions for Arcadia Financial’s Managed Clients still go through a legacy workflow application that the firm has yet to migrate off of. Not only has the application’s vendor been charging insane amounts for extended support, the organization has largely abandoned the product completely, meaning that no additional integrations/updates are being made outside of sev 1 security updates. Each day, as Arcadia Financial advisors work with clients to perform rollovers, transfers, or IRA contributions - the transactions are logged into the FinTech WorkFlow Solutions platform. The back office operations team of Arcadia Financial are assigned cases based on their security clearance and familiarity with Arcadia products, to transfer data from the FinTech Workflow Solution’s interface to specific workflow transactions in Arcadia's more modern Case Management platform for validation and processing. One of these validations is to ensure that the correct financial advisor is listed on the case. Often, the associate generating the data fails to update the advisor on the case, so before migrating the data, you will need to append the name of the newest advisor to any unlabelled cases: “Jennifer Allston”.

Challenge Goal

Create an embedded automation solution to migrate all FIVE rows of data from the Legacy Application Portal (linked below) to the Modern Case Management Platform (also linked below). Once completed, click “Validate Submission” to finish the challenge and receive your score.

Tip

Transactions are randomized every time you load the challenge page, so make sure the transactions are read from the live page before you look for a matching transaction in the banking application. When you complete the challenge, a GUID will show up at the bottom of the results modal. Copy and save the GUID’s for your accuracy score

IMPORTANT Use the “HINT” links on the Legacy Portal to get helpful hints on how to use the new AARI Extensions!

Rules

You are expected to launch the both portal from the challenge page and then login to the app. Also, do not cache the transaction details from the banking application since that’s not how it is done in the production environment, and we want to mimic a similar experience for this challenge. You must integrate AARI extensions to perform a portion or all of the challenge.

Conclusion

Swivel chair data migration is an extremely popular use case in the world of Automation. This challenge is meant to show you the new AARI Extensions features in a use case which everyone is comfortable. Embedded automation can be a powerful tool in your toolkit when tackling the human / automation interaction. From embedding bot trigger buttons on pages, to bringing context and information into the applications where users reside, we hope this new feature will enable more powerful and seamless automations!

Like all our challenge pages, there’s more than one way to solve this – so even if you build it in a certain way, play around with different approaches to see how that impacts the reliability of your automation and the usefulness for an end user.

Finally, if you tried this challenge and are getting stuck, don’t worry! We’ll be demonstrating the solution during a Tech Talk at our Imagine conference. Best of luck! Go Be Great!
